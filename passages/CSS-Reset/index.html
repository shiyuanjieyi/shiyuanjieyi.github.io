里面写的是自己定制简单的CSS重置样式和一些理解说明，我觉得挺有趣的，下面我就把它里面写的东西用中文重写一遍

其实也就只有9条，非常简短，但是它们很有用，下面我就一一解释一下：

1.我们学过css的都应该了解过盒子模型，如下图：

box-sizing: content-box 是默认的方式，它是指给容器设置的 width 和 height 仅仅只包含 Content 那部分；
box-sizing: border-box 是指给容器设置的 width 和 height 出了 Content 那部分，还包含 Padding 和 Border 这两部分；

在日常开发过程中，我们一般给容器设置宽高度都是要包含那两部分的，因此需要给它重置一下。

2.删除默认外边距

浏览器对页边距做出常识性的假设。例如，h1默认包含的页边距比段落多。但是通常情况下，我发现自己希望元素默认没有任何元素。所以我决定把它们全部移除。

3.设置以百分比计算的身高

其实我在做项目过程中，还会将宽度设为100%，设高度为100%反而比较少，在移动端居多。

4.调整行高

这里设置行高标准说是1.5倍，然而我在微信小程序样式规范上提倡是1.4倍。

5.字体平滑

这个属性之前我没用过，看作者举的例子，也没看出太大的区别，所以这个有点争议。

6.明智的媒体默认设置

设置为块级元素能够避免内联元素产生的像素问题，我还设置了最大宽度: 100% 。这样做是为了防止大图像溢出，如果它们被放置在容器中，容器不够宽以容纳它们。

7.继承窗体控件的字体

我记得之前在做H5输入框时，经常会遇到点击输入框，整体页面会放大，这里我之前是用input的聚焦事件强制禁止缩放来实现的，现在有这个新方案会简洁得多。

8.文字包装

这里实现英文和数字能够强制换行。

9.根堆叠上下文

最后一个是可选的，通常只有在使用 JS 框架时才需要，比如 React，隔离属性允许我们创建一个新的堆叠上下文，而不需要设置 z-index。
